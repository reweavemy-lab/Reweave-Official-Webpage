# Reweave Authentication System

## Live Authentication System

This is a standalone authentication system for Reweave - Luxury Batik Fashion.

### Features
- ✅ User Registration
- ✅ User Login  
- ✅ Session Management
- ✅ Account Dashboard
- ✅ Responsive Design
- ✅ Luxury Brand Styling

### How to Use
1. Click "Sign Up" to create a new account
2. Fill in your details (Name, Email, Password)
3. Click "Create Account"
4. You're instantly logged in!

### Technical Details
- Pure HTML/CSS/JavaScript
- Client-side authentication
- LocalStorage session management
- No backend required
- Mobile responsive

### Files
- `netlify-account.html` - Main authentication page
- Standalone system that works immediately
